# Resume Chat

## Prompts

### Basics

- Get name
- Get email
- Get phone number
- Get address (optional)
- Get information for the summary

### Experience

Repeat these prompts until the user is done adding previous jobs.

- Get company name
- Get job title
- Get location
- Get dates worked
- Get basic work tasks (optional)

### Education

Repeat these prompts until the user is done.

- Get college
	- Or college (optional)
- Get location
- Get years studied
- Get major
- Get GPA (optional)

### Skills

Repeat these prompts until the user is done.

- Get skills

### Certifications

Repeat these prompts until the user is done.

- Get certification
- Get date
